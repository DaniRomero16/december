Votación

Este portal servirá como método alternativo para el ejercicio de derecho a voto disponible a partir de 2019 para todo ciudadano que lo solicite.

Miembros de Mesa

Si has sido elegido como miembro de mesa a través de este portal puedes realizar todas las gestiones necesarias, además de encontrar toda la información que necesite para la jornada electoral.
